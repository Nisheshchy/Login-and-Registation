# Premium Auth Frontend

A modern, high-performance authentication UI built with React and Vite. This project features a stunning glassmorphism design, smooth animations, and a responsive layout.

## Features
- **Premium UI**: Modern dark-themed dashboard with glassmorphism and Tailwind CSS.
- **Micro-animations**: Smooth transitions using Framer Motion.
- **Local Authentication**: Persisted user accounts in `localStorage`. Users must **Sign Up** before they can **Log In**.
- **Admin Access**: Hardcoded Admin credentials for system management.
- **Protected Routes**: Frontend logic for dashboard access.
- **Responsive Design**: Fully optimized for mobile and desktop screens.

## Folder Structure
- `src/`: React frontend source code.
- `public/`: Static assets.

## Prerequisites
- Node.js (v18 or higher)
- npm or yarn

## Setup Instructions

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
The application will start on `http://localhost:5173` (default Vite port).

## Technical Stack
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7
